What is HTML?

Hypertext Markup Language (HTML) is the standard markup language for documents that are designed to be displayed in a web browser. HTML is not a programming language. It is a markup language that tells browser how to structure a web page.

A sime HTML document looks like this

<!DOCTYPE html> <html lang="en"> <head> <title>My page</title> </head> <body> <h1>My page</h1> <p>This is a <a href="link">simple</a> Link</p> <!-- this is a comment --> </body> </html> As in the above example, html document consists of a tree of elements and text. Each element starts with a tag b> and ends with a tag. Tags have to be nestedb> such that they are completely within each other without overlapping .
